## 3. Main Findings

### Finding 1: Existing Website Is Strong, But Area Mismatch Exists

The current website is visually strong and service-focused, but it positions Jhapher around Cagayan de Oro, Misamis Oriental, Misamis Occidental, Cotabato City, and Mindanao dispatch. That is useful proof of capability, but it may confuse NCR/Rizal customers if reused without explanation.

**Opportunity:** Create a dedicated NCR/Rizal/Greater Manila service-area version or landing page set.

### Finding 2: Facebook Has Useful Public Proof, But Access Is Friction-Heavy

The Facebook post screenshot shows services and contact numbers, but Facebook is partly gated by login modals and public-view limitations.

**Opportunity:** Use Facebook as social proof, but make the website the official, easy-to-read hub.

### Finding 3: Google Business Profile Needs Manual Confirmation

The Maps screenshot did not confirm a clean owned listing for the NCR/Pasig query.

**Opportunity:** Verify or create/optimize GBP as a service-area business, with hidden address if needed and true service areas only.

### Finding 4: Competitors Already Cover NCR / Provincial Service Areas

Competitor pages show structured service pages, phone numbers, service-area links, and broad coverage claims. This means customers already see more organized competitors when searching.

**Opportunity:** Build a cleaner, more trustworthy local service structure for Jhapher, with real service areas, actual photos, and easy quote/call CTAs.
