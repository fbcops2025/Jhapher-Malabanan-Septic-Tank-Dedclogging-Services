Booking guide · Practical guide

Most delays in the first conversation come from missing basics: where, what fixtures, since when, and how bad. A complete first message saves time.

Lead with location

City, barangay, and a landmark. Add subdivision, building, or gate notes if they affect entry.

Name the affected fixtures

Toilet only? Kitchen sink? Floor drain? Several bathrooms? Be specific.

Describe the symptom in plain words

Slow, fully blocked, gurgling, water returning, odor, overflow. Short and direct is best.

Add timing and change

When it started, whether it is worsening, and whether it followed heavy use, guests, or rain.

Attach safe photos when possible

Access point, standing water, or the affected fixture helps. Avoid unsafe close contact with wastewater.

Use this simple message shape

Location + affected fixtures + symptoms + timing + access photo. That five-part message is usually enough to start.

Send your request

Five-part request checklist

  • Exact location
  • Fixture list
  • Symptom words
  • Start time and changes
  • Safe photo
